# Test Task App

A demo application built with **Vite**, showcasing a modular architecture.

---

## 🚀 Getting Started

### Install dependencies
```bash
npm install
```

### Run in development mode
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

## 🛠️ Requirements
**Node.js** >= 22.19.0 <br>
**npm** >= 10.9.3
